# cryptpw

## ⚠️ Warning

> **This project is currently in early development and is NOT secure.**  
> Do **not** use it to store real passwords or sensitive information.  
> Security features are incomplete and have not been audited.


## Description

**cryptpw** is an experimental password manager written in Rust.

It currently acts as a local client, with plans to support multiple user profiles for managing separate sets of credentials.

The project is still in its early stages, with minimal functionality, and is mainly focused on exploring secure storage, encryption practices, and Rust-based system design.
